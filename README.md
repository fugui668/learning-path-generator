# 🎯 个性化学习路径生成器

根据个人学习目标、当前水平、可用时间，自动生成有序学习计划 + 资源推荐 + 进度检验方案。

## 快速开始

```bash
# 交互式生成（推荐）
python3 learning_path.py

# 查看3个示例
python3 learning_path.py --demo

# 进度追踪 + 动态调整建议
python3 learning_path.py --track
```

**无需安装任何依赖，仅需 Python 3.6+**

---

## 功能说明

### 1. 信息收集
交互式输入 4 项信息：
- **学习目标**（自然语言描述，自动识别领域）
- **当前水平**（零基础 / 初级 / 中级 / 高级）
- **每周可用时间**（小时）
- **期望完成周期**（周数）

### 2. 学习路径生成
- 根据领域（编程/数据分析/英语/通用）匹配对应阶段模板
- 按当前水平决定起始阶段（入门/进阶/高级）
- 按时间约束自动压缩/扩展每步周数（比例限制在 0.5x~2x）
- 输出有序步骤 + 里程碑 + 总学时

### 3. 资源推荐类型
每个阶段推荐 4 类资源，按阶段差异化：
| 阶段 | 核心资源类型 |
|------|------------|
| 入门 | 入门书籍、视频课程、练习题、学习社群 |
| 进阶 | 进阶文档、动手项目、专项平台、技术博客 |
| 高级 | 论文/源码、竞赛/开源、技术分享、同行交流 |

### 4. 进度检验（掌握度评估）
每阶段 3 个检查点：
- **概念自测**：能否用自己的话解释核心概念？
- **实操验证**：练习题正确率达标？
- **成果展示**：完成阶段里程碑任务？

### 5. 动态调整建议
`--track` 模式输入落后周数，自动给出分级调整方案：
- **落后 1 周**：加密度、跳扩展
- **落后 3 周**：重规划里程碑、降深度要求
- **落后 5 周+**：重评目标、从已完成点重启

## 路径生成逻辑说明

### 需求拆解方式
需求拆解为 4 层独立模块：输入收集 → 领域检测 → 路径生成 → 输出格式化。每层单独测试，最后组合。

### AI 协作方式
- 阶段模板（知识点顺序、里程碑定义）由 AI 根据领域最佳实践生成
- 自适应时间分配算法（按比例压缩/扩展）由 AI 设计并验证边界情况
- 检验标准（掌握度自测题目）由 AI 根据教育心理学原则设计

### 关键取舍
| 取舍项 | 选择 | 原因 |
|--------|------|------|
| 实现方式 | CLI（非 Web/App） | 零依赖，快速可运行，专注核心逻辑 |
| 资源推荐 | 类型推荐（非真实链接） | 避免链接失效，普适性更强 |
| 领域覆盖 | 编程/数据/英语/通用 | 4 类覆盖 80% 场景，避免过度设计 |
| 路径算法 | 规则引擎（非 AI API） | 本地运行，无网络依赖，输出稳定可预期 |
